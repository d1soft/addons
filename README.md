# openode addons

Official addons on [opeNode.io](https://www.openode.io/).

# config.json schema

| Variable        | Description |
| ------------- |:-------------:|
|  name     | Addon name. Must be unique. |
|  category     | Category (folder) of the addon |
| minimum\_memory\_mb | Minimum memory required (MB) |
